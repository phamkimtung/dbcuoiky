## Common Workflows

### Creating a New Student

1. **Navigate to Students Tab**
   - Click "Students" in tab list
   - Table loads with current students

2. **Click "Add Student" Button**
   - Form dialog appears

3. **Fill Required Fields**
   - Full Name
   - Email
   - Password
   - Student Code (e.g., SV001)
   - Select Major from dropdown
   - Admission Year

4. **Click "Create"**
   - API call creates user + student record
   - Success toast appears
   - Table refreshes automatically
   - Dialog closes

### Updating Student Information

1. **Find student in table**
2. **Click "Edit" button**
3. **Form pre-fills with current data**
4. **Modify fields** (except password on edit)
5. **Click "Update"**
6. **Confirmation toast** + table refresh

### Deleting a Student

1. **Find student in table**
2. **Click "Delete" button**
3. **Confirm in dialog** - "Are you sure?"
4. **Student + user account deleted**
5. **Success message** + table refresh

### Creating a Class Schedule

1. **Navigate to Classes Tab** вҶ’ Create Class
2. **Fill form:**
   - Select Course (dropdown)
   - Select Lecturer (dropdown)
   - Select Semester (dropdown)
   - Enter Max Students

3. **Go to Schedules Tab** вҶ’ Add Schedule
4. **Assign schedule to created class:**
   - Select Class
   - Choose Day of Week
   - Enter Start Time
   - Enter End Time
   - Enter Room

5. **Confirmation** and table updates

## Tips & Tricks

1. **Refresh Button** - Click to reload data from server
2. **Search** - Currently available for students only
3. **Scroll** - Tables scroll horizontally on small screens
4. **Confirmation** - Dialogs confirm before destructive operations
5. **Status Badges** - Color-coded for quick visual scan
